
# FrogPad

 A 3x3 (9 keys) macropad designed by me for personal use. I use this as a simple shortcut map for my desk setup. I chose the color green because I like frogs.


## Features:

This design comes with an EC11 Rotary encoder for volume control, and 9 keyswitches with customization available
